# Formulário de Cadastro

Projeto desenvolvido de forma prática através de um desafio focado em estruturas de formulários da plataforma Rocketseat. A aplicação consiste em um sistema de cadastro para a criação de convites digitais personalizados.

## Funcionalidades

*   **Foco em UX:** Estrutura organizada seguindo princípios de usabilidade e acessibilidade para facilitar o preenchimento de dados pelo usuário.
*   **Design Clean:** Estilização moderna e minimalista utilizando propriedades avançadas de CSS.
*   **Interface Semântica:** Uso correto das tags HTML5 para validação de campos nativa e melhor leitura do navegador.

## Tecnologias Utilizadas

*   **Front-end:** HTML5 e CSS3
